## Development


Para comenzar a trabajar en el proyecto:

1. Clonar este repositorio

```shell
git clone url
```

2. Después, ingresar al directorio o carpeta del repositorio.

```shell
cd Coding Challenge
```

3. Instalar las dependencias de `node`:

```shell
npm install
```

4. Iniciar el servidor de desarrollo:

```shell
npm start
```
